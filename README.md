# 倒數計時器

一個功能完整的網頁倒數計時器，具有音效提醒和自動循環功能。

## 功能特色

- ⏰ **自定義倒數時間**：可設定任意秒數的倒數時間
- 🔔 **提醒功能**：可設定在剩餘特定秒數時發出音效提醒
- 🎵 **鈴聲音效**：自動生成的鈴聲音效，持續約3秒
- ▶️ **控制功能**：開始、暫停、重置按鈕
- 🔄 **自動循環**：倒數結束後自動重新開始，直到用戶暫停或重置
- 📊 **視覺回饋**：進度條顯示和時間變色提醒
- 📱 **響應式設計**：支援手機和桌面裝置

## 使用方法

1. **設定倒數時間**：在"倒數時間"欄位輸入總秒數（預設60秒）
2. **設定提醒時間**：在"提醒時間"欄位輸入要提醒的秒數（預設10秒）
3. **開始倒數**：點擊"開始"按鈕
4. **控制倒數**：
   - 點擊"暫停"暫停倒數
   - 點擊"重置"重置到初始狀態
5. **音效提醒**：當剩餘時間等於設定的提醒時間時，會播放鈴聲

## 技術特色

- **純 JavaScript**：不依賴任何外部函式庫
- **Web Audio API**：動態生成鈴聲音效
- **現代 CSS**：使用 CSS Grid 和 Flexbox 布局
- **音效備援**：如果無法播放音效，會嘗試使用瀏覽器通知

## 瀏覽器支援

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## 使用須知

1. **音效權限**：首次使用時瀏覽器可能會要求音效播放權限
2. **通知權限**：如果音效無法播放，會請求瀏覽器通知權限
3. **背景運行**：頁面需要保持在前台才能正常播放音效

## 開始使用

直接在瀏覽器中開啟 `index.html` 即可使用。
